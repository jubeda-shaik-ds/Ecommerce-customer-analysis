# 🌍 E-Commerce Customer Analysis

## 📌 Project Overview

This project analyzes customer behavior and sales distribution using multiple e-commerce datasets. It focuses on understanding customer locations and their contribution to sales.

---

## 🎯 Objective

* Analyze customer distribution
* Identify top states by sales
* Understand customer behavior

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib

---

## 📂 Dataset

* Orders
* Payments
* Customers

---

## 🔍 Key Analysis

* Sales by state
* Customer count
* Data merging across multiple datasets

---

## 📈 Results

* Identified top-performing states
* Analyzed customer distribution
* Gained insights into regional sales

---

## 🚀 How to Run

1. Install libraries:
   pip install pandas matplotlib

2. Run:
   python analysis.py
